Documentación de Vercel

¿Qué es Vercel?

Vercel es una plataforma en la nube optimizada para aplicaciones web modernas, desde sitios estáticos hasta apps impulsadas por inteligencia artificial (IA).

Funciones Principales

Inicio Rápido

  • Crea y aloja apps con tu framework favorito.
  • Conecta tu repositorio Git para despliegue automático.
  • Usa entornos de vista previa antes de producción.

Construcción de Aplicaciones

  • Next.js y otros frameworks compatibles.
  • API Routes y Funciones para lógica del servidor.
  • Middleware, regeneración estática, optimización de imágenes.
  • Gestión de entornos y banderas de características.

Infraestructura de IA

  • v0: Asistente de desarrollo con IA.
  • SDK de IA y AI Gateway para integración de modelos.
  • Agentes IA, servidores MCP y entornos sandbox.
  • Implementaciones de reclamaciones para control humano.

Colaboración

  • Barra de herramientas para comentarios, edición en vivo, etc.
  • Comentarios en implementaciones.
  • Modo borrador para CMS.

Seguridad

  • Protección de implementaciones y acceso (RBAC).
  • WAF configurable, gestión de bots y DDoS.
  • BotID y filtrado de bots IA.

Escalabilidad y Despliegue

  • Red global de entrega de contenido.
  • Versiones continuas y reversión rápida.
  • Observabilidad y monitoreo de rendimiento.
